# Per-scenario stderr/stdout substring contract.
# Format:
#   <scenario_id> <stream> <+|-> <substring...>
# stream: stderr | stdout
# +  -> substring MUST appear at least once
# -  -> substring MUST NOT appear
# Lines starting with `#` and blank lines ignored.
# Quote-free: the substring runs to end-of-line (trailing spaces trimmed).
#
# Catches "rc=0 but message regressed" — invisible to expected_rc.txt.
# Add an entry whenever a CLI message is the actual user-visible contract
# (errors, hints, warnings whose absence would be a regression) or when a
# noisy log was *intentionally* removed/deduped (negative check pins it).
#
# Regenerate after intentional message changes; document in CHANGELOG.
